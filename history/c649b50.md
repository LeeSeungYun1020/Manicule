# c649b50 - core:ui 모듈에 BookProgressBar 및 ContributionGrid 추가

- **WHY**: Track B "6-2. core:ui (기타)" 진행.
- **의사결정**:
  - `ContributionGrid` 초기 로드시, 사용자가 가장 최신 데이터를 즉시 볼 수 있도록 가장 우측 끝단을 기본 스크롤 위치로 지정 (reverseLayout 대신 초기 인덱스로 ScrollState 설정)
  - 월요일이 제일 위에 오도록 하기 위해, 첫 데이터 날짜의 요일을 기반으로 앞에 빈(empty) 셀 패딩을 추가하여 정렬 구조 유지
  - 더미 데이터를 사용하는 `ContributionPreviewParameterProvider`를 별도로 분리하여 향후 Stats 화면 등에서도 함께 사용할 수 있도록 구성
