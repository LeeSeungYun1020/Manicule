## ReadingRecord — 누적 페이지 (cumulativePage)

### 결정
하루 읽은 페이지가 아닌 **그 날까지의 누적 페이지** 저장.

### 이유
- "4월 24일 — 100쪽까지" 사용자 멘탈 모델과 일치.
- 진도 = `MAX(cumulativePage) WHERE isbn = ?` 단순 쿼리.
- record 누락 / 순서 바뀜에도 진도 보존. 당일 페이지는 인접 record 차분.
- Invariant: monotonic 증가 → Repository 가 음수 차분 reject.
