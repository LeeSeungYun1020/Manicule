# f830ee6 — build: local.properties에서 NLK_AUTH_KEY 미설정 시 빌드 오류 발생

**Files**: `core/network/build.gradle.kts`

## 결정
- `NLK_AUTH_KEY`가 `local.properties`에 없는 경우 빌드 시점에 오류(`GradleException`)가 발생하도록 설정

## 이유
- 해당 값이 없을 경우 기본값 `""`이 들어가면서 빌드는 성공하지만 런타임에 실패하거나 불필요한 오류가 발생하게 됨
- 개발자가 API Key 누락 사실을 즉시 알아차릴 수 있도록 빌드 시점에 차단하여 방어적 개발 적용
