# 5942347 — ai: Network/Image BoM 전역 적용 제거

**Files**: `AndroidCommon.kt`

## 결정
`configureKotlinAndroid()` 가 OkHttp / Retrofit / Coil BoM 을 모든 Android 모듈에 자동 적용하던 것을 제거. 헬퍼 함수도 삭제.

## 이유
- `core:common`, `core:designsystem` 등 비-네트워킹 모듈에 platform 제약 전파 → 의존성 그래프 혼탁.
- BoM 은 "필요한 모듈에서 명시적으로 선언" 이 원칙.
- 대안: `manicule.android.library.network` / `.coil` 같은 dedicated convention plugin. 현재 사용 모듈 1~2 개라 over-engineering — 3+ 모듈로 늘면 재검토.
- 적용 시점: `core:network` 만들 때 `platform(libs.okhttp.bom)` / `platform(libs.retrofit.bom)` 직접 선언, `core:ui` 만들 때 `platform(libs.coil.bom)` 직접 선언.
