# 읽음표

국어: 읽음표 - 나의 독서 기록
영어: Manicule — A Reader's Mark

## 1. 컨셉

읽은 책, 읽고 있는 책, 읽고 싶은 책을 한 곳에서 관리하는 개인 서재 앱.
국립중앙도서관 API로 국내 도서 서지 정보를 검색하고, 독서 상태와 메모를 기록한다.

---

## 2. 타겟 사용자

- 읽은 책을 기록해두고 싶은데 마땅한 앱이 없는 독서 애호가
- 독서 목표를 세우고 달성 여부를 확인하고 싶은 사람
- 서점에서 책을 발견했을 때 나중에 읽으려고 메모해두는 사람

---

## 3. 핵심 기능

### 3.1 홈

- 검색창 — 탭 시 검색 화면으로 이동 (키보드 자동 포커스)
- 바코드 스캔 버튼 — 탭 시 바코드 스캔 화면으로 이동
- 읽는 중인 책 목록 (섹션 헤더 우측 "내 서재 →" 버튼 탭 시 서재 탭으로 이동)
  - 읽는 중인 책이 없을 때: "읽는 중인 책이 있나요? 지금 책을 추가해 보세요." 메시지와 함께 검색 유도 (탭 시 검색창에 포커스)
- 독서 잔디 미리보기 (최근 8주), 연속 기록 일수(streak, 일일 누적 페이지 ≥ 1인 날을 연속 카운트) — 탭 시 통계 화면으로 이동
- 오늘 요약 통계 (오늘 읽은 페이지 수) — 탭 시 통계 화면으로 이동

### 3.2 책 검색

- 제목, 저자, ISBN으로 국내 도서 검색
- 검색 결과: 표지 이미지, 제목, 저자, 출판사, 발행일
- 최근 검색어 저장 및 재사용
- 검색 결과 없는 경우: "바코드가 있나요? 바코드를 스캔해 보세요" 안내와 함께 바코드 스캔 화면으로 이동
- ISBN 바코드 스캔으로 즉시 검색 (카메라 권한 필요)
  - 스캔 실패 시 공통 다이얼로그로 "ISBN 인식에 실패하였어요" 표시 후 검색 화면으로 이동

### 3.3 책 상세

- 표지, 책 소개, 목차, 출판 정보 (페이지 수, 가격 등)
- 독서 상태 설정 (읽고 싶음 / 읽는 중 / 완독)
- 별점(1~5 정수) 및 한 줄 메모 기록
- 독서 기록: 날짜별로 읽은 누적 페이지를 기록하여 진도 추적
  - 예: "4월 24일 — 100쪽까지" (100쪽까지 읽었다는 의미)
  - 통계 계산 시 이전 기록과의 차이로 당일 읽은 페이지 수 산출
  - 읽는 중 상태에서 최근 기록을 상단에 노출
  - 기록이 쌓이면 날짜별 독서 히스토리 조회 가능
  - 기록 수정 및 삭제 가능
  - 읽고 싶음 상태에서 기록 추가 시 읽는 중으로 자동 전환
  - 완독 상태에서는 읽는 중으로 변경 후 기록 가능
  - 기록 후 남은 페이지(전체 - 현재 기록)가 40쪽 이하이면 공통 다이얼로그로 "혹시 책을 다 읽으셨나요?" 표시, "네" 선택 시 완독으로 자동 전환
- 독서 상태 전환 규칙
  - 완독으로 변경한 날짜를 완독일로 저장
  - 읽은 페이지와 무관하게 완독으로 변경 가능(페이지 계산에는 실제 읽은 페이지만 반영)

### 3.4 내 서재

- 상태별 탭: 전체 / 읽고 싶음 / 읽는 중 / 완독
- 읽는 중 탭: 카드에 현재 페이지 / 전체 페이지 진도 표시 (예: 132 / 320쪽)
- + 버튼으로 검색 화면 진입하여 책 추가
- 정렬: 수정 날짜 순 / 추가 날짜 순 / 별점순
- 책 삭제 시 공통 다이얼로그로 "책에 기록한 내용이 모두 삭제되어요" 확인 후 삭제
- 텝/정렬 기본값: 읽는 중, 수정 날짜 순

### 3.5 독서 통계

- **기간 전환**: 이번 달 / 올해 단위로 전환하여 요약 통계 조회
  - 완독한 책 수
  - 읽은 페이지 수
- **독서 잔디**: 지난 365일을 52주 × 7일 그리드로 표시 (가로 스크롤), 하루 읽은 페이지 수에 따라 5단계 intensity로 색상 강도 표현 (0 / 1–19 / 20–49 / 50–99 / 100+ 페이지). 날짜 칸 탭 시 해당 날의 독서 기록을 잔디 하단에 표시

### 3.6 설정

- 다크 / 라이트 / 시스템 테마
- 독서 리마인더 알림 (on/off, 알림 시간 설정)
  - 컨텍스트 기반 메시지: 가장 최근 '읽는 중' 책 제목을 포함, 없으면 일반 메시지로 fallback

## 4. 화면 목록

> 하단 탭: 홈 / 서재 / 통계 / 설정

| 화면 | 진입 경로 | 주요 내용 |
|---|---|---|
| 홈 | 하단 탭 | 검색창, 바코드 스캔 버튼, 읽는 중 책, 잔디 미리보기, streak, 오늘 요약 통계 |
| 검색 | 홈 검색창 또는 서재 + 버튼 | 검색창, 최근 검색어, 결과 목록 |
| 바코드 스캔 | 홈 바코드 스캔 버튼, 검색 결과 없는 경우 | 카메라 뷰파인더, ISBN 인식 후 책 상세로 이동, 실패 시 다이얼로그 후 검색 화면으로 이동 |
| 책 상세 | 검색 결과 또는 서재에서 탭 | 표지, 소개, 상태/별점/메모 입력, 독서 기록(날짜+페이지) |
| 서재 | 하단 탭 | 상태별 책 목록, + 버튼 |
| 통계 | 하단 탭 또는 홈 카드 탭 | 52주 잔디, 이번 달/올해 요약 통계 |
| 설정 | 하단 탭 | 테마, 리마인더 알림 |

---

## 5. 유저 플로우

```
[홈 탭]
  → 검색창 탭 → 검색 화면으로 이동
  → 바코드 스캔 버튼 탭 → 바코드 스캔 화면으로 이동
  → 읽는 중 책이 없을 때: Empty State 메시지 탭 -> 검색창에 포커스
  → 읽는 중 책 탭 → 책 상세로 이동
  → 읽는 중 섹션 "내 서재 →" 버튼 탭 → 서재 탭으로 이동
  → 잔디 미리보기 / streak / 오늘 통계 카드 탭 → 통계 탭으로 이동(필요하다면 위치로 스크롤)

[검색 화면]
  → 최근 검색어 탭 → 해당 검색어로 검색 실행
  → 검색어 입력 → 결과 목록에서 책 선택 → 책 상세로 이동
  → 검색 결과 없는 경우 → "바코드가 있나요? 바코드를 스캔해 보세요" → 바코드 스캔 화면으로 이동
  
[바코드 스캔 화면]
  → ISBN 인식 성공 → 책 상세로 이동
  → ISBN 인식 실패 → "ISBN 인식에 실패하였어요" 다이얼로그 → 검색 화면으로 이동

[책 상세]
  → 독서 상태 설정 (읽고 싶음 / 읽는 중 / 완독) → 서재에 자동 추가
  → 독서 기록 추가
    → 읽고 싶음 상태: 기록 추가 시 읽는 중으로 자동 전환 후 저장
    → 읽는 중 상태: 기록 저장, 수정, 삭제 가능
      → 남은 페이지 40쪽 이하 → "혹시 책을 다 읽으셨나요?" 다이얼로그
        → "네" → 완독으로 전환, 완독일 저장
        → "아니요" → 기록만 저장
    →  완독 상태: 기록 추가, 수정, 삭제 불가
      → 읽는 중으로 상태 변경 → 읽는 중 상태로 기록 추가, 수정, 삭제 가능
  → 읽는 중 → 완독으로 상태 직접 변경 → 완독일 저장

[서재 탭]
  → 탭으로 상태 필터
  → 정렬 버튼 탭 → 정렬 옵션 선택 (수정 날짜 순 / 추가 날짜 순 / 별점순) → 목록 재정렬, 선택값 유지
  → 책 탭 → 책 상세로 이동
  → 삭제 버튼 → "책에 기록한 내용이 모두 삭제되어요" 확인 다이얼로그 → 삭제
  → + 버튼 탭 → 검색 화면으로 이동

[통계 탭]
  → 잔디에서 날짜 칸 탭 → 잔디 하단에 해당 날의 독서 기록 표시
  → 이번 달 / 올해 전환으로 요약 통계 조회

[설정 탭]
  → 테마 선택 (다크 / 라이트 / 시스템) → 즉시 적용
  → 독서 리마인더 알림 on/off 토글
    → on: 알림 시간 설정 → 매일 해당 시간에 알림 발송
    → off: 알림 비활성화
```

## 6. 데이터 소스

**국립중앙도서관 ISBN 서지정보 API** (data.go.kr 신청, 무료, 상업적 이용 가능)

활용 필드:
- 표제, 저자, 출판사, 발행일
- 표지 이미지 URL
- 책 소개, 목차
- 페이지 수, 가격, 분류 (한국십진분류)
- ISBN
