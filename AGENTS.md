# 참고 파일

- plan/plan.md 기획
- plan/structure.md 모듈과 파일 구조
- plan/order.md 작업 순서, PR로 리뷰 가능한 수준으로 단계를 더 쪼개어 구현 가능
- plan/current.md 작업한 내용, 다음 작업 내용. 작업 시 plan/current.md 파일 업데이트하여 다음 세션에서 활용
- history/ 커밋별 의사결정 기록 (작성 규칙: history/README.md)
- skills/skills.md 스킬 파일. 사용자가 코드 구현, PR 생성, 코드 리뷰, 리뷰 반영, 리뷰 검수를 요청하면 해당 스킬 파일 절차를 준수하여 실행.
